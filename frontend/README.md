# Веб-интерфейс (Frontend) для сервиса прокси-доступа

Этот модуль содержит пользовательский интерфейс для управления доступом к прокси-серверам и взаимодействия с FastAPI бэкендом.

## Технологический стек

* **Фреймворк:** Vue 3 (Composition API)
* **UI-библиотека:** Vuetify 3
* **Стейт-менеджер:** Pinia
* **Маршрутизация:** Vue Router
* **Сборщик:** Vite
* **HTTP-клиент:** Axios

---

## Как запустить проект локально (режим разработки)

Для локального запуска требуется установленный Node.js.

1. Перейдите в директорию фронтенда (если вы находитесь в корне):
   ```bash
   cd frontend
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите сервер для разработки:
   ```bash
   npm run dev
   ```

После запуска интерфейс будет доступен в браузере (обычно по адресу http://localhost:3000).

---

## Сборка для продакшена (Docker)

В корневом файле `docker-compose.yml` всего проекта уже настроена автоматическая сборка этого модуля. При запуске команды `docker-compose up` проект компилируется и раздается через веб-сервер Nginx (конфигурация описана в файле `Dockerfile`). В этом случае сайт будет доступен по стандартному порту: http://localhost.
